# 一期一会ノベル（仮）

アプリを開くと、静かな本棚の画面。
まだ何も並んでいない、少しだけ余白のある空間。

「マッチング開始」を押すと、
どこかの誰かと、ゆるやかにつながる。

■ 出会いは、名前も声もない

集まるのは2〜4人。

相手が誰なのかは分からない。
年齢も性別も、性格も知らない。

あるのは、同じ物語をこれから共有する“参加者”という関係だけ。

わずかな緊張と、少しの期待。

■ 物語が“その場で”生まれる

カテゴリーが決まると、
画面に静かに文字が浮かび上がる。

AIが、今この瞬間のためだけに書く導入文。

ゆっくりと紡がれていく文章を、
全員が同時に見守る。

「この物語は、今ここでしか読めない」

そんな実感が、じわじわと広がる。

■ 沈黙の中で、選択だけが交差する

物語が進むと、突然——選択肢が現れる。

残り10秒。

言葉は交わせない。
相談もできない。

ただ、それぞれが“どうしたいか”を選ぶ。

助けるのか
見捨てるのか
進むのか、引き返すのか

他の誰かが何を選ぶのかは分からない。

それでも、どこかで“人の気配”を感じる。

■ 多数決が、空気をつくる

結果が表示される瞬間。

自分と同じ選択をした人がいたのか、
それとも、裏切られたのか。

2対2で割れる緊張。
思いもよらない多数派。

そして、時には——
突然現れる“想定外の第3の選択肢”。

「こんな展開になるのか」

プレイヤー全員が、同じ驚きを共有する。

■ “誰かと一緒に読んでいる”感覚

会話はない。
リアクションも最小限。

それでも、

同じタイミングで驚いている
同じ文章を読んでいる
同じ結末に向かっている

という、不思議な一体感が生まれる。

まるで無言の読書会のような、
静かで濃い共体験。

■ エンディングの余韻

物語が終わると、
ゆっくりと本が閉じられる。

少しだけ残る余韻。

「この物語、どうする？」

■ 最後の選択——残すか、消えるか

その物語を保存するかどうかも、また多数決。

残したいと思う人
“この一度きり”で終わらせたい人

全員の意思がぶつかる。

可決された場合だけ、次へ進む。

■ タイトルをつける瞬間

それぞれがタイトルを考える。

同じ物語を読んだはずなのに、
出てくる言葉はまったく違う。

そして、また多数決。

選ばれたひとつのタイトルが、
その物語の“正式な名前”になる。

■ 本棚に残る、あるいは消える

保存された物語は、本棚に並ぶ。

あのときの選択、あの結末。
あの“誰かたち”との時間。

逆に否決されれば——
その物語は、どこにも残らない。

誰の記録にも残らず、ただ記憶の中だけにある。

■ プレイ後に見える“自分”

最後に表示される、選択傾向。

あなたはリスクを取るタイプか
感情で動いたのか
他人とどれだけ一致したのか

自分では気づかなかった“物語の選び方”が、
少しだけ可視化される。

この体験の本質

このゲームは、

「上手くプレイするゲーム」ではなく、
「どう選ぶかを体験するゲーム」。

そして同時に、

「誰かと一緒に、同じ物語を“違う気持ちで読む”ゲーム」。
